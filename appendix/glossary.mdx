---
title: "Glossary"
description: "Thuật ngữ chính trong AhaOps Copilot"
---

| Thuật ngữ | Giải thích ngắn |
| --- | --- |
| **Core Copilot** | Lớp dùng chung để hiểu intent, quản lý context, gọi tool và handoff |
| **Skill Pack** | Gói nghiệp vụ riêng của một team: knowledge, workflow, tools, guardrails, evaluation |
| **MCP Tool** | Chức năng được expose theo schema để Agent gọi API / service nội bộ có kiểm soát |
| **Workflow Engine** | Nơi lưu trạng thái case, approval và transition |
| **RBAC** | Phân quyền dựa trên vai trò |
| **Handoff** | Chuyển case sang owner phù hợp cùng thông tin đã được tóm tắt |
| **Source of Truth** | Nguồn dữ liệu / policy chính thức được dùng để xác minh |
| **Evaluation Set** | Bộ test kiểm tra Agent trước khi release |
